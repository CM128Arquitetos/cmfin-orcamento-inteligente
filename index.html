Configurações Financeiras

Alterações pendentes de salvamento

Fórmula base por dias trabalhados: Diária útil = Horas p/ dia útil × Hora trabalhada útil. Diária sábado = Horas p/ dia sábado × Hora trabalhada sábado. Valor do Projeto = (Dias úteis × Diária útil) + (Dias sábados × Diária sábado).

Taxa de estrutura da empresa: aplicada sobre o valor do projeto para formar caixa de estrutura, ferramentas, marketing, melhorias e novos investimentos.

Rateio do custo fixo: Custo fixo semanal = Custo fixo mensal ÷ 4. O sistema considera a quantidade estimada de semanas do projeto para incluir uma parcela do custo fixo em cada orçamento.

Margem alvo: margem ideal para o projeto ser saudável, pagar seu trabalho, cobrir custos e deixar lucro real.

Contingência padrão: reserva aplicada sobre os custos para cobrir retrabalho, reuniões extras, ajustes, imprevistos e variações de escopo.

Os campos desta tela são editados manualmente. Após alterar qualquer valor, clique em Salvar configurações para aplicar os novos parâmetros no orçamento.
